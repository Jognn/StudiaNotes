## ELF
Exectuable and Linkable Format - jest standardem do plików exectuable

**Jest używany do:**
- Executables
- Shared libraries
- Objects
- Core dumps

 Podział na *segments* (runtime) i *sections* (link time)

#### Sections
- `.text` - code
- `.rodata` - read only data
- `.data` - read/write data
- `.bss` - zero-initialized data
